# Портфолио (локальный сервер)

Текущие файлы `index.html` и `style.css` отдаются на `http://localhost:8080`.

## Запуск

1. Открой PowerShell в папке `c:\QA\Cursor`
2. Выполни команду:

   ```powershell
   node server.js
   ```

3. Открой в браузере:

   `http://localhost:8080`

## Остановка

Заверши процесс в терминале (Ctrl+C).
